# Sovereign Bitcoiner Workshop - Code Orange Dev School

## Overview

This repository contains slides and materials from the **Sovereign Bitcoiner Workshop** presented by Code Orange Dev School at Bitcoin Amsterdam 2025. This comprehensive workshop is designed for developers and Bitcoin enthusiasts who want to understand Bitcoin from both theoretical and practical perspectives.

## Workshop Content

### Core Topics Covered

- **Bitcoin Fundamentals**: Introduction to Bitcoin, blockchain technology, and distributed systems
- **Decentralization & Sovereignty**: Understanding financial independence and self-custody principles
- **Security & Self-Custody**: Best practices for securing Bitcoin, managing private keys, and using hardware wallets
- **Bitcoin Development**: Technical deep-dive into transactions, scripting, and the Bitcoin protocol
- **Lightning Network**: Layer-2 solutions for scalability and fast payments
- **Practical Implementation**: Hands-on guidance for developers building Bitcoin applications

### Key Learning Areas

1. **Blockchain Fundamentals**
   - How Bitcoin works at a technical level
   - Consensus mechanisms and proof-of-work
   - Transaction mechanics and validation

2. **Security & Best Practices**
   - Private key management
   - Hardware wallet usage
   - Common security pitfalls and how to avoid them
   - Self-custody principles

3. **Development & Implementation**
   - Bitcoin transaction creation and signing
   - Smart contracts and Bitcoin scripting
   - Lightning Network development
   - Developer tools and frameworks

4. **Philosophy of Sovereignty**
   - Why Bitcoin matters for financial freedom
   - Decentralization principles
   - Real-world applications and case studies

## Audience

This workshop is ideal for:
- Developers interested in Bitcoin development
- Bitcoin enthusiasts wanting deeper technical knowledge
- Anyone interested in financial sovereignty and self-custody
- DevOps professionals exploring blockchain technology

The slides are organized to progress from foundational concepts to advanced implementation details, with practical examples and real-world use cases throughout.
